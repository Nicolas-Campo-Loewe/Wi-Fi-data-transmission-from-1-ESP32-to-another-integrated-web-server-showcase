# Wi-Fi data  transmission from 1 ESP32 to another + integrated web server showcase

Brief description
- Wi-Fi connection between an ESP32 server and an ESP32 client. Basic telemetry data (temperature shown in UTC time) is sent and displayed on the web server via the ESP32 client.
